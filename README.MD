# Assembly of God Church - Ruwanwella Website

Modern church website built with HTML, CSS, and JavaScript

## Features
- Responsive design
- Welcome page with hero section
- Our story page with church history
- Ministries section (Men's, Women's, Youth, Children)
- Service times display
- Online donations section
- Contact form

## Technologies Used
- HTML5
- CSS3 (Flexbox, Grid)
- JavaScript (ES6)
- Google Fonts
- Font Awesome Icons
